# Cursor Landing Page UI Clone
A desktop only pixel-perfect black theme cursor clone build using only HTML and CSS.\
No libraries or Frameworks used.\
<br>
🔗 Live Demo: (https://cursor-cloned-ui.netlify.app/)
<br>
## 🏗️ Recreated Sections
The Project contains the following recreations:
- Sticky Navigation
- Hero Section
- Trust Gallery
- Product Feature Articles
- Testimonials
- Global Footer\
<br>

## 🎨 Design Tokens 
**Font** : Helvetica Neue, Helvetica, Arial, sans-serif\
<br>
## 🛠️ Built With

* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Flexbox for alignment, `position: sticky` for navigation, and custom hover states.

<br>

## 🏁 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/cursor-clone.git](https://github.com/your-username/cursor-clone.git)
    ```

2.  **Open the project:**
    Simply open `index.html` in your browser.
